       Kiến trúc kỹ thuật công nghệ
Kiến trúc công nghệ xác định các nền tảng, công nghệ, tiêu chuẩn kỹ thuật cụ thể được lựa
chọn để xây dựng, triển khai và vận hành Hệ thống thông tin quản lý kết cấu hạ tầng giao
thông hàng hải (CSDL KCHTGTHH).
Kiến trúc công nghệ được chia thành các lớp chính sau:

   -   Lớp Trình diễn (Presentation Layer):

           o Giao diện Web: Người dùng truy cập hệ thống chủ yếu thông qua trình duyệt
               web tiêu chuẩn (Chrome, Firefox, Edge, Safari phiên bản mới nhất) trên máy
               tính cá nhân hoặc thiết bị di động. Giao diện được thiết kế theo chuẩn
               Responsive Web Design để đảm bảo hiển thị tốt trên các kích thước màn
               hình khác nhau.

           o Công nghệ: HTML5, CSS3, JavaScript.

           o Framework Frontend: Sử dụng framework ReactJS để xây dựng giao diện
               người dùng tương tác phong phú, hiệu quả và dễ bảo trì.

           o Thành phần Bản đồ (GIS): Tích hợp thư viện bản đồ nền web Leaflet để hiển
               thị, tương tác với dữ liệu không gian, kết nối với dịch vụ bản đồ từ lớp ứng
               dụng.

   -   Lớp Ứng dụng (Application Layer):

           o Nền tảng backend: Sử dụng nền tảng Java (với Spring Boot framework) để
               xây dựng các dịch vụ backend.

           o Kiến trúc: Áp dụng kiến trúc microservices để đảm bảo tính module hóa, khả
               năng triển khai và nâng cấp độc lập.

           o Giao tiếp API: Sử dụng RESTful API với định dạng dữ liệu JSON làm
               phương thức giao tiếp chuẩn giữa Frontend và Backend, cũng như giữa hệ
               thống với các ứng dụng/hệ thống bên ngoài.
